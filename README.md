## Installation

### Clone

- Clone this repo to your local machine using:
```shell
$ git clone https://github.com/ArthurPaes/Github-Explorer.git
```

### Setup


> Execute `yarn` or `npm instal` to install all the dependencies

```shell
$ yarn 
$ npm install
```
